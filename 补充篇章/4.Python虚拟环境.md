# Python虚拟环境

## 操作步骤

1. 树莓派上的python2和python3

2. 为什么需要python虚拟环境

3. 创建python虚拟环境

4. 在虚拟环境中安装老版本home-assistant

5. 运行虚拟环境中的home-assistant

6. 升级虚拟环境中的home-assistant

## 参考

-  Python虚拟环境参考文档

    https://virtualenv.pypa.io/en/stable/

- Python虚拟环境命令

    - 创建虚拟环境（xxxx是创建的虚拟环境目录名）

        `python3 -m venv xxxx`

        或

        `virtualenv xxxx`

    - 进入虚拟环境

        `source 虚拟环境目录/bin/activate`

    - 退出虚拟环境

        `deactivate`
