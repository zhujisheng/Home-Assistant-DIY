# Arduino与ESP硬件

## 操作步骤
1. 下载安装Arduino IDE

    https://www.arduino.cc/en/Main/Software

2. 增加esp8266与esp32开发库

3. 连接ESP硬件，选择对应的硬件平台，选择COM口
4. arduino编程语言，setup和loop函数
5. 编写一个每秒闪烁一次led的固件程序

    ```c
    void setup() {
      pinMode(2, OUTPUT);
    }

    void loop() {
      digitalWrite(2, HIGH);
      delay(500);
      digitalWrite(2, LOW);
      delay(500);
    }
    ```
